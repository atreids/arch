# Commands

## Useful CLI commands

- `btop` View general system info and running processes
- `neofetch` View OS info
- `lsd` - Nicer ls
- `fzf` - Fuzzy find

## Custom

- `copy-ghostty-term` custom script to copy Ghostty xterm terminal info to a remote server via (and for) SSH. Place in `~/.local/bin`.
